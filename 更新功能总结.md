  🚀 部署和使用流程

  1. 环境评估

  # 运行探测工具评估IPv6环境
  ./ipv6-probe.exe

  # 生成推荐配置
  ./ipv6-probe.exe --gen-config > ipv6-config.json

  2. 配置启用

  // config.json
  {
    "mode": "local",
    "userName": "MyNode",
    "enableIPv6": true,
    "ipv6Mode": "preferred",
    "udpEnabled": true,
    "encryptionMode": "noise"
  }

  3. 启动应用

  # 启动IPv6增强版本
  go run main_integrated.go -config config.json

  # 或编译后运行
  go build -o p2p-chatroom-ipv6.exe .
  ./p2p-chatroom-ipv6.exe
